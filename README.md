# Neural-GARCH 模型

使用神经网络增强的 GARCH 模型，用于金融时间序列的条件波动率预测与风险管理。

---

## 模型简介

Neural-GARCH 模型结合了传统 GARCH 模型的统计结构与神经网络的非线性建模能力。  
不同于传统 GARCH 的线性参数形式，Neural-GARCH 使用神经网络来刻画条件方差的递归演化：

\[
\sigma_t^2
= f_\theta\bigl(r_{t-1}^2,\ \sigma_{t-1}^2\bigr)
\]

其中：

- \( r_t \) 为去均值后的收益率
- \( \sigma_t^2 \) 为条件方差
- \( f_\theta(\cdot) \) 为带参数 \( \theta \) 的神经网络

---

## 核心特点

- ✅ **保留 GARCH 的金融解释性**：维持条件波动率的递归结构  
- ✅ **非线性建模能力**：神经网络捕捉复杂的波动率动态  
- ✅ **概率预测**：输出完整条件分布，而非点预测  
- ✅ **风险度量支持**：可计算 VaR、CVaR（ES）等指标  
- ✅ **轻量级设计**：参数规模可控，降低过拟合风险
